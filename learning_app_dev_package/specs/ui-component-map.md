# UI 组件映射

| 组件 | 用途 |
|---|---|
| AppTopBar | 模块切换、目录按钮、设置入口 |
| SidebarTree | 笔记目录、题库目录、错题本镜像目录 |
| ContextMenu / MobileBottomSheet | 桌面右键与手机长按操作 |
| NoteEditor | CodeMirror 费曼笔记编辑器 |
| QuestionBankView | 章节列表和题目列表 |
| QuestionFormModal | 创建和编辑结构化题目 |
| PracticeSetupModal | 选择来源范围、顺序、目标错题本和名称 |
| MemorizeCard | 认识、不认识、重点 |
| QuizRunner | 单题全屏刷题 |
| PracticeReport | 每轮刷题结果和下一轮入口 |
| WrongBookMirrorTree | 根据原题章节动态生成镜像结构 |
| BackupRestorePanel | 完整备份、恢复和存储状态 |
| VersionHistoryPanel | 笔记历史快照预览与恢复 |
