# 题目规则与判定规范

## 单选题

- 选择后立即判断。
- 答错：错误选项红色，正确选项绿色；加入目标错题本。
- 答对：可按设置自动跳转。

## 多选题

- 支持多个选项选中和取消。
- 点击“提交答案”后判断。
- 正确集合必须完全一致。
- 内部统一将 `ACD`、`A,C,D`、`A C D` 解析为数组。

## 判断题

- 接受：对 / 错、正确 / 错误、true / false、√ / ×。
- 内部统一为 boolean。

## 填空题

- 题干中的连续下划线转换为输入框。
- 多个空使用 `||` 分隔答案。
- 同义答案使用 `|` 分隔。
- 默认忽略首尾空格、大小写和连续空格差异。

## 简答题

- 不进行自动文本评分。
- 用户显示答案后，手动点击“我会了”或“不认识”。

## 通用“不会”按钮

- 所有题型提供“不会”。
- 点击后立即显示答案和解析，记录为 `unknown`。
- 若已配置目标错题本，按原题位置添加引用。
