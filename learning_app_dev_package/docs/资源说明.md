# 资源说明

本资料包中的 UI 界面稿、图标、PWA 图标和插图均为本次需求文档配套生成的原创矢量资源，可直接用于原型和开发参考。

- `ui/svg/`：可编辑 SVG 界面稿。
- `ui/png/`：对应 PNG 预览。
- `assets/icons/`：基础 SVG 图标。
- `assets/logo/`：PWA 应用图标，包含 SVG、192px PNG 和 512px PNG。
- `assets/illustrations/`：空状态、离线可用和定期备份插图。

这些界面稿用于说明布局和交互结构，并非最终视觉定稿。开发时可根据实际品牌风格继续微调字号、间距、颜色和动画。
